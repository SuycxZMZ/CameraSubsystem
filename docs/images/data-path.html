CameraSubsystem 数据路径

Camera HW → V4L2 采集 → BufferPool → IPC 分发 → Gateway → WebSocket → 浏览器

采集链路

  • • V4L2 MMAP 模式从 Camera HW 采集
  • • CameraSource 适配层封装平台差异
  • • BufferPool 管理 FrameHandle 生命周期
  • • 当前为拷贝模式，DMA-BUF 零拷贝待打通

IPC 分发

  • • Data IPC: Unix Domain Socket
  • • CameraDataHeader (帧头) + 帧数据
  • • 复制分发，每个订阅端独立接收
  • • 发送失败自动移除断连客户端

Web 推送

  • • Gateway 转换为 WebFrameHeader (92B)
  • • WebSocket binary frame 推送
  • • JPEG 原始透传，无服务端编码
  • • 浏览器 createImageBitmap + Canvas 渲染

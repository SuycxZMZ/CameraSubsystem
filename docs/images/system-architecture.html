CameraSubsystem 系统架构

Camera 数据流基座 — 发布/订阅模型 + Web Preview 扩展

核心发布端

  • • 独占 Camera 设备，V4L2/MMAP 采集
  • • SessionManager 按订阅引用计数启停
  • • Control IPC: Subscribe / Unsubscribe / Ping
  • • Data IPC: Unix Socket 帧头 + 帧数据

Web Preview 扩展

  • • Gateway 作为订阅端接入核心发布端
  • • HTTP 静态服务 + WebSocket 二进制帧推送
  • • WebFrameHeader 92B 协议对齐 C++ struct
  • • React Canvas 渲染 JPEG，支持多流

下游订阅端

  • • AI 推理、视频编码、录制等
  • • 通过 Control IPC 订阅/退订
  • • 通过 Data IPC 接收帧数据
  • • 可扩展 FrameTransformPipeline

Camera Codec Server Architecture

CameraSubsystem board-side H.264 recording path with raw preview preserved.

Preview Boundary

  • • Web preview keeps raw stream subscription
  • • Browser never receives recording file data
  • • Gateway is a control proxy for recording

Recording Path

  • • camera_codec_server subscribes original stream
  • • USB first uses JPEG decode before H.264
  • • MIPI later uses DMA-BUF import path

Hardware Choice

  • • RK3576 VPU supports H.264/H.265 encode
  • • Rockchip MPP is the preferred encoder API
  • • Software encoder is fallback only
